## Troubleshooting

**Q: Script won't run?**
```bash
# Make it executable
chmod +x quick_research_contacts.py
```

**Q: Missing pandas?**
```bash
# For AI version only
pip install pandas openai
```

**Q: CSV not uploading?**
- Check file size (max 10MB)
- Verify CSV format matches template
- Make sure you selected "Supply Contracts" type
