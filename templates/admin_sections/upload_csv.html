CSV Format Guide
Required Columns:

Local/State Government Contracts:

  • title
  • agency
  • location
  • value
  • deadline
  • description
  • url
  • posted_date (optional)

Federal Contracts:

  • title
  • agency
  • location
  • value
  • deadline
  • description
  • url (SAM.gov URL)
  • notice_id
  • naics_code
  • posted_date (optional)

Supply Contracts:

  • title
  • agency
  • location
  • product_category
  • estimated_value
  • bid_deadline
  • description
  • website_url
  • contact_name (optional)
  • contact_email (optional)
  • contact_phone (optional)
  • is_quick_win (true/false)
  • status (open/closed)
  • posted_date (optional)
The first row must contain column headers matching these exact names.
