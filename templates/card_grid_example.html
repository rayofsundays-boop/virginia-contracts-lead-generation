How to Use

Copy this HTML structure for your cards:

<div class="card-grid">
    <div class="uniform-card">
        <div class="uniform-card-body">
            <div class="uniform-card-badge">Optional Badge</div>
            <h3 class="uniform-card-title">Card Title</h3>
            <p class="uniform-card-description">
                Your description text here...
            </p>
        </div>
        <div class="uniform-card-footer">
            <a href="#" class="uniform-card-btn">Button Text</a>
        </div>
    </div>
</div>

Features:

  • ✅ All cards maintain equal height
  • ✅ Buttons perfectly aligned at bottom
  • ✅ Smooth hover effects
  • ✅ Fully responsive (4 cols → 3 cols → 2 cols → 1 col)
  • ✅ Works with varying content lengths
  • ✅ Optional badges and icons
  • ✅ Primary and secondary button styles
